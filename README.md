# Suman Shrestha Research Website

A static academic website designed for GitHub Pages.

## Files

- `index.html` — homepage
- `research.html` — research overview
- `projects.html` — project summaries
- `documents.html` — CV, talks, and notes
- `style.css` — visual design
- `script.js` — mobile navigation and current year

## Publish with GitHub Pages

1. Create a public GitHub repository named:

   `suman0725.github.io`

2. Upload all files from this folder into the repository root.

3. In GitHub, open:

   `Settings → Pages`

4. Choose:

   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`

5. Open:

   `https://suman0725.github.io`

## Before publishing

Replace these placeholders:

- `your-email@temple.edu`
- ORCID link
- LinkedIn link
- Profile-photo placeholder
- CV file

To add your photo, place a file such as `profile.jpg` in the repository and replace the photo placeholder in `index.html` with:

```html
<img class="profile-photo" src="profile.jpg" alt="Suman Shrestha">
```

You may also add this CSS:

```css
.profile-photo {
  width: 100%;
  border-radius: 14px;
  display: block;
}
```

## Important

Do not publish:

- internal CLAS12 documents
- unpublished collaboration results
- restricted data
- private emails
- passwords, tokens, or credentials
- personal visa or identification documents
